# 網頁版俄羅斯方塊 3D 版

這是一個使用 **HTML + JavaScript (Canvas)** 製作的網頁版俄羅斯方塊，支援 3D 立體方塊、速度提升、計分、下一個方塊預覽等功能。  
只要打開 `tetris.html` 即可直接遊玩，不需要伺服器。

---

## 🔧 功能特色

- 3D 立體方塊（高光 + 陰影效果）
- 標準 10×20 棋盤
- 7 種方塊（I / O / T / J / L / S / Z）
- 自動下落 + 硬降（Space）
- 消行計分、等級提升機制
- 顯示下一個方塊
- 可暫停 / 重新開始
- 支援鍵盤操作

---

## 🎮 操作說明

| 按鍵 | 功能 |
|------|------|
| ← → | 左右移動 |
| ↓ | 加速下落 |
| ↑ | 旋轉方塊 |
| 空白鍵 | 硬降（直接掉到底） |
| P | 暫停 / 繼續 |
| R | 重新開始 |
